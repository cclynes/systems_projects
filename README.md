Computer Systems Projects
===============

This repository contains five projects inspired by assignments in Brown's Computer Systems class. Stencil code was cloned from the Github repositories. I completed these projects independent of the class, out of interest in computer systems and a desire to get a rigorous standard introduction to the fundamentals of computer systems.

The projects are:
- Snake (introduction to C programming)
- DMalloc (memory allocator for debugging)
- FileIO (file I/O cache)
- WeensyOS (virtual memory and syscalls for tiny operating system)
- Concurrent Store (thread-safe key-value store)

I didn't write these projects with the intention of sharing them, but if you do want to run them locally, let me know and I'll add some info about the container and running.

For more information about the class these projects were assigned in:
http://cs300.systems/

### What I wrote
Because these were class projects, much of the stencil code, including tests, were not written by me. This allowed me to focus on learning systems concepts without getting bogged down in setup details, but only code committed by me should be taken as mine. On the other hand, since the purpose of these projects was learning, I consulted AI for some design and conceptual questions but all of my code was written by hand with my full understanding.
